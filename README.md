# AiPowerHouse UI

Multi-model Command Center for orchestrating AI workflows.

## Features

- **Unified AI Hub**: Build, compare, and deploy across models.
- **Model Roster**: Support for ChatGPT, Claude, Gemini, and more.
- **Routing Playbooks**: Automated switching and safety checks.
- **Safety & Governance**: Guardrails, PII redaction, and compliance.

## Getting Started

### Prerequisites

- Node.js 20+
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Testing

```bash
npm test
```

### Building

```bash
npm run build
```
